# WhatWasThat Browser Extension

Browser companion for the WhatWasThat desktop app.

## Setup
1. Open desktop app settings
2. Copy API token from Security section
3. Load extension in developer mode
4. Paste token in extension options

## Features
- Save page screenshots
- Save image URLs
- Save text selections
- Authenticated API requests
